# Island Borne Website — Phase 4

Complete Odoo 19 Island Borne website module built from the supplied brand source files.

Pages:
- Home (`/`)
- About (`/about`)
- Products (`/products`)
- Fragrances (`/fragrances`)

Includes the reusable branded header and footer, responsive mobile menu, favicon, locally hosted Afacad and Vampiro One fonts, product artwork, responsive store finder and direct-loaded CSS/JavaScript for Cloudpepper.
