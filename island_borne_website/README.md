# Island Borne Website — Phase 2

A fresh Odoo 19 website module created from the supplied Island Borne designer source package and the approved homepage corrections.

## Included

- Approved responsive homepage served at `/`
- New branded About page served at `/about`
- Reusable global Island Borne header and footer
- Header and footer About links connected to the new page
- Afacad body typography and Vampiro One display typography
- Original Island Borne logo, icon, scent icons and product artwork
- Original tropical source photography optimized for responsive web use
- Working mobile burger navigation
- Interactive desktop and mobile store finder map
- Responsive single-image footer artwork
- Island Borne favicon
- Direct-loading CSS and JavaScript for Cloudpepper reliability

## Upgrade

Replace the existing `island_borne_website` add-on folder, restart Odoo, update the Apps list and upgrade **Island Borne Website**.
